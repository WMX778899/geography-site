# 首页图标抛入动画说明

## 🎬 动画效果

页面加载时，各个图标会以轻柔的"抛入"效果依次出现，营造生动活泼的视觉体验。

## ✨ 动画特点

### 1. **温和不夸张**
- 起始位置：从上方15px处开始（不是30px）
- 轻微旋转：最大2度（几乎看不出的倾斜）
- 缩放变化：从0.9到1.02（非常轻微的大小变化）
- 弹性效果：使用cubic-bezier(0.34, 1.56, 0.64, 1)创造轻微回弹

### 2. **层次分明的时序**
```
0.1s  → Logo图标 🌍
0.2s  → 统计数字
0.3s  → 第一个模块卡片图标 🌋
0.4s  → 第二个模块卡片图标 🏙️
0.45s → 第三个模块卡片图标 🗺️
0.5s  → 第四个模块卡片图标 🌱
0.55s → 搜索按钮 🔍
0.6s  → 知识地图、标签等小元素
0.85s → 页脚
```

### 3. **覆盖的元素**

#### 首页
- ✅ Logo图标
- ✅ 统计数据数字
- ✅ 四个模块卡片图标
- ✅ 搜索按钮图标
- ✅ 高频考点标签
- ✅ 备考指南图标
- ✅ 知识地图标题
- ✅ 频道横幅按钮
- ✅ 学习提示图标
- ✅ 页脚

#### 知识页面
- ✅ 章节图标（🪐、🌤️、🌊等）
- ✅ 侧边栏标题
- ✅ 侧边栏导航链接
- ✅ 侧边栏频道卡片

## 🎨 动画参数

```css
/* 关键帧 */
0%   →  opacity: 0, translateY(-15px), rotate(-2deg), scale(0.9)
60%  →  opacity: 1, translateY(3px), rotate(1deg), scale(1.02)
80%  →  translateY(-2px), rotate(-0.5deg), scale(0.99)
100% →  translateY(0), rotate(0), scale(1)

/* 缓动函数 */
cubic-bezier(0.34, 1.56, 0.64, 1)

/* 持续时间 */
0.35s - 0.6s（根据元素类型不同）
```

## 🔄 重复播放

- **首次加载**：自动播放
- **页面切换**：每次切换页面都会重新触发动画
- **JavaScript重置**：通过`triggerPageAnimations()`函数重置所有动画

## 📱 移动端支持

- ✅ 动画在移动设备上流畅运行
- ✅ 不影响触摸交互
- ✅ 不会造成性能问题（使用transform和opacity）

## 🎯 设计理念

1. **轻柔优雅**：运动幅度小，不过分张扬
2. **自然有序**：元素按逻辑顺序依次出现
3. **品牌呼应**：绿色主题贯穿始终
4. **用户体验**：增添活力但不干扰内容浏览

## 🚀 技术实现

### CSS动画
```css
@keyframes iconDropIn { ... }
animation: iconDropIn 0.4s cubic-bezier(...) forwards;
animation-delay: 0.1s;
```

### JavaScript重置
```javascript
function triggerPageAnimations() {
    // 重置所有动画元素的animation属性
    // 强制重排后重新应用动画
}
```

## 📝 注意事项

- 动画使用`forwards`关键字，保持最终状态
- 所有元素初始`opacity: 0`，动画结束后可见
- 避免使用`display`或`visibility`属性（会影响动画）
- 使用`transform`而非`top/left`（性能更好）
