# 知识页面描述文字可读性优化

## 问题描述

四个知识页面的描述文字在页面上几乎看不清：

- **自然地理**: "研究自然环境各要素的形成、发展、分布及其相互关系的规律"
- **人文地理**: "研究人类活动与地理环境的关系，探讨人地协调与可持续发展"
- **区域地理**: "认识中国与世界各区域的地理特征，理解区域差异与联系"
- **选修专题**: "自然资源、环境保护、灾害防治与区域可持续发展"

## 问题原因

### 1. **背景色对比度不足**
```css
/* 原始样式 */
.page-header p {
    background: rgba(46, 139, 87, 0.08);  /* 透明度太低 */
    border: 1px solid rgba(46, 139, 87, 0.2);
    opacity: 0.95;
}
```

**对比度分析**：
- `.page-header p` 背景：`rgba(46, 139, 87, 0.08)` - 极淡的绿色
- `.page--nature .main` 背景：`linear-gradient(..., rgba(46, 139, 87, 0.05), ...)` - 同样是淡绿色
- **结果**：文字背景和页面背景几乎相同，导致文字可读性极差

### 2. **透明度问题**
- `opacity: 0.95` 虽然数值接近1，但叠加其他半透明元素后进一步降低了对比度

### 3. **缺少视觉层次**
- 没有阴影效果，无法将文字标签与背景分离

## 解决方案

### ✅ **优化策略**

1. **加深背景色** - 从 0.08 提升到 0.22
2. **加深边框** - 从 0.2 提升到 0.45
3. **增强阴影** - 添加柔和阴影增加层次感
4. **统一不透明度** - 设置为 1，移除 opacity
5. **增强字体** - 添加 font-weight: 500 和 line-height: 1.6
6. **主题色适配** - 每个页面使用对应主题色的深色版本
7. **悬停效果** - 添加微交互提升体验

### 📊 **对比数据**

| 属性 | 优化前 | 优化后 | 改善效果 |
|------|--------|--------|----------|
| 背景透明度 | 0.08 | 0.22 | ↑ 175% |
| 边框透明度 | 0.2 | 0.45 | ↑ 125% |
| 阴影 | 无 | 0 2px 10px | 新增层次 |
| 不透明度 | 0.95 | 1.0 | 消除叠加损失 |
| 字体粗细 | 默认 | 500 | 更清晰 |
| 行高 | 默认 | 1.6 | 更易阅读 |


- 悬停时轻微上浮
- 阴影加深，增加立体感
- 过渡动画 0.3s ease

## 技术实现

### CSS 修改点

1. **基础样式提升**
   - 背景色：`rgba(46, 139, 87, 0.08)` → `rgba(30, 107, 63, 0.22)`
   - 边框色：`rgba(46, 139, 87, 0.2)` → `rgba(30, 107, 63, 0.45)`
   - 增加 `box-shadow` 和 `transition`
   - 设置 `opacity: 1` 和 `font-weight: 500`

2. **页面主题色适配**
   - 为 `.page--nature`、`.page--humanity`、`.page--region`、`.page--elective` 分别定义样式
   - 每个使用对应主题色的深色变体
   - 文字颜色统一为 `#1a1a1a`（接近纯黑）

3. **悬停效果**
   - 每个页面类型都有独立的悬停样式
   - 背景加深 + 阴影增强 + 上浮动画

4. **移动端适配**
   - 调整字体大小和内边距
   - 确保小屏幕上的可读性
